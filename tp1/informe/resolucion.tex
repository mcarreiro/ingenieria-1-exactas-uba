\subsection{Calificaciones} %Agregra una subseccion por tema
\begin{itemize}
\item Un viaje es un recorrido instanciado según una fecha. Es decir, el recorrido indica qué días se va a realizar, y el sistema detecta cuando se cumplió otro período del recorrido y crea un viaje.
\item Una calificación se asigna a un conductor a partir de un viaje.
\item La calificación del viaje indica si el mismo se realizó o no
\end{itemize}

\subsection{Retribuciones} %Agregra una subseccion por tema
\begin{itemize}
\item Un viaje se considera pago cuando los pasajeros y el conductor informaron que pagaron y recibió el pago respectivamente.
\end{itemize}
